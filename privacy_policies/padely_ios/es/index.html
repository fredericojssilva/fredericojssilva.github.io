Política de Privacidad

Frederico Silva desarrolló la aplicación Padely como una aplicación gratuita. Este SERVICIO se proporciona sin coste alguno y se ofrece tal cual.

Esta página se utiliza para informar a los usuarios sobre las políticas relativas a la recopilación, uso y divulgación de información personal.

Recopilación y Uso de Información

La aplicación no recopila, almacena ni comparte datos personales identificables.

La aplicación puede realizar solicitudes de red para obtener contenido. Estas solicitudes pueden incluir información técnica como la dirección IP, gestionada por los protocolos estándar de Internet y no almacenada por la aplicación.

Bibliotecas y Tecnologías de Terceros

  • Kotlin Multiplatform (KMP)
  • Ktor
  • Coil 3
  • AndroidX / Jetpack
  • Jetpack Compose

Estas tecnologías no recopilan datos personales.

Datos de Registro

En caso de error, pueden generarse datos técnicos básicos del sistema. Esta información no es recopilada ni almacenada por la aplicación.

Cookies

Esta aplicación no utiliza cookies ni tecnologías de seguimiento.

Almacenamiento de Datos

Los datos se almacenan únicamente en el dispositivo del usuario.

Seguridad

Aunque se aplican buenas prácticas, ninguna transmisión es completamente segura.

Enlaces Externos

La aplicación puede contener enlaces a sitios externos, cuya política no controlamos.

Privacidad Infantil

No se recopilan datos de menores de 13 años.

Cambios

La política puede actualizarse periódicamente.
